# Learning about Socket.IO

Used in this project:
1. TypeScript
2. Next.js
3. Express.js
4. Awilix
5. Awilix-express
6. Socket.IO
7. SASS (7-1 pattern)

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

To test chat on single machine you might use incognito and usual tab in same time or different browsers.
